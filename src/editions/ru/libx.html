Privacy:

By using this extension you may send the URL of pages you are currently visiting to the Radford University McConnell Library catalog server. Such information is sent only if you are actively using the extension; it is never sent automatically. radford University's privacy policy states that it may use this information to improve its services.
If you wish to avoid this you must turn off the referrer URL by setting network.http.sendRefererHeader to 0 in your preferences.
